Toy Automatic Differentiation
=============================

Introduction
------------


The `repository <https://github.com/CyberZHG/toy-auto-diff>`_ is a **TOY** demonstration of how to write automatic differentiations based on `numpy <https://www.numpy.org/>`_.


Install
-------

.. code-block:: bash

   pip install toy-auto-diff
